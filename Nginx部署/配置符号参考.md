## 配置符号参考

容量符号缩写

| k,K  | 千字节 |
| ---- | ------ |
| m,M  | 兆字节 |

例如, "8k", "1m" 代表字节数计量。

时间符号缩写

| ms   | 毫秒        |
| ---- | ----------- |
| s    | 秒          |
| m    | 分钟        |
| h    | 小时        |
| d    | 日          |
| w    | 周          |
| M    | 一个月,30天 |
| y    | 年,365天    |

例如, "1h 30m", "1y 6M". 代表 "1小时 30分", "1年零6个月"。
